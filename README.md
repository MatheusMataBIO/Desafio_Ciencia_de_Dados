## Projeto de Ciências de dados  

### Objetivo 🎯 

Esse projeto tem como objetivo praticar minhas habilidades minhas habilidades em ciência de dados como praticar treinamento de modelos de ML

### Descrição

Nesse case temos que fazer a previsão do preço de apartamento.

Todo desenvolvimento e o script foi criado no google colaboratory

### Dados

Os dados foram extraídos de um dataset fornecido para o desafio

### Execução

Abra o arquivo com script no repositório e execute célula por célula
